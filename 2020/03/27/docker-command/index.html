常见场景

基于当前镜像下载某些依赖生成新的镜像

使用场景: 某些环境机器不能访问外网，如果镜像中需要下载外部依赖，则会失败，这时可以在可以访问外网的机器基于当前镜像下载好相应依赖再生成新的镜像

操作步骤:

  • 编写 Dockerfile
  • 在可访问外网的机器构建新镜像
  • 将镜像保存到文件中
